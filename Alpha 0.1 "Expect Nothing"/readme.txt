Hatch-Broswer
A different Python based browser.

VERSION: Alpha 0.1 "expect nothing"

Brief Windows Instruction: --install python3-- winget install Python.Python.3.14 https://www.python.org/downloads/windows/ click on "Download Installer (MSIX)" under the Stable Releases it will download to your downloads folder. double click on the file "python-manager-26.3", or the version might be a higher number. after this runs, open the power shell and type python3 and hit enter.

--install pip-- follow the youTube video "How to Install PIP in Python - Windows 10/11 (2024) by The Code Bear https://pip.pypa.io/en/stable/installation/

--install parakimo-- just reference: "https://www.paramiko.org/installing.html" pip install paramiko

--install tk-- should work out of the box
